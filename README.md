# ase25-assignment04-bt724930
Yet another homework assignment.
Run the main file in Smart Cater and interact with the terminal from there.

The system can query ingredients and meals. You can also set labels, which act as a filter for meals i. e. if you select "BIO", the searched product will only be shown if it has this label. The ingredient and meal enum files can be extended easily by adding new enums. The label enum file too.
